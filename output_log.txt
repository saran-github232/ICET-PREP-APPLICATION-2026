# Plan: Configure for GitHub Pages Deployment

This plan prepares the project for hosting on **GitHub Pages**. Note that this is different from Vercel; GitHub Pages is often used for static sites and requires specific configuration for React Single Page Applications (SPAs).

## User Review Required
> [!IMPORTANT]
> **Why this change?** 
> GitHub Pages requires a `base` path (e.g., `/ICET-PREP-APPLICATION-2026/`) to load assets correctly, and a GitHub Action to automate the build.
> 
> **SPA Routing Caveat**
> GitHub Pages doesn't natively support SPA routing. If you refresh a page like `/select`, it will return a 404. I will implement a common "404 hack" to handle this, but using **HashRouter** is the most reliable alternative if the hack doesn't suit your needs.
